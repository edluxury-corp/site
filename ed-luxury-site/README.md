# ED Luxury — landing page

Première version statique fidèle à la maquette crème.

## Ouvrir le site
Ouvrir `index.html` dans un navigateur.

## Remplacer les images
Les fichiers se trouvent dans `assets/` :
- `birkin-hero-placeholder.svg` : Birkin principal détouré
- `selection-placeholder.svg` : sélection de sacs
- `showroom-1.svg`, `showroom-2.svg`, `showroom-3.svg` : appartements haussmanniens

Tu peux remplacer chaque SVG par un JPG, PNG ou WebP et modifier simplement le chemin dans `index.html`.

## Couleurs principales
- Fond : `#f6f6f6`
- Noir : `#191714`
- Doré : `#b59661`
